# TOOLS.md - Local Notes

Record setup-specific details that help the agent use your tools correctly.

## What Belongs Here

- Device names and nicknames
- Camera locations
- SSH hosts
- Preferred voices or speakers
- Local service URLs
- Any other environment-specific tool notes

## Example

```markdown
### SSH

- home-server -> 192.168.1.100, user: admin

### Devices

- office-speaker -> Desk speaker in the upstairs office
```
